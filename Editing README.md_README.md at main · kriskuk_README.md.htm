        ###

        <p align="left">Я начинающий специалист по обеспечению качества (QA) с опытом ручного тестирования и активно изучаю автоматизацию на Python. Моя история началась с неудачной попытки сдать экзамен по химии и биологии и продолжается с успешному перепрофилированию на IT. Наконец я занимаюсь тем что мне реально нравится и чему я готова постоянно учиться.

        В начале своего обучения я познакомилась с тем как выглядит сайт изнутри и за что отвечает каждый его кусочек, продолжая обучаться я узнала о немаловажной и, по моему мнению, очень важной роли в разработке любого проекта - тестировщик. Пройдя несколько курсов на Stepik и YouTube, я по-настоящему очаровалась этой профессией.

        Сейчас я в активном поиске позиции Junior QA. Мне нравится находить сложные баги и делать продукты лучше.</p>
